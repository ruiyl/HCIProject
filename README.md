# HCIProject
This project was developed as a final project for Human Computer Interaction, Virtual and Augmented Reality course. The project is an AR table tennis game using functionalities from Vuforia and is built for mobile devices. All the code comments are for the grading purpose.
## Installation
- Vuforia package is needed for the project to build and run, but may not be included automatically by Unity on the first time the project is loaded.
- Default camera needs to be selected for Editor Play Mode. Edit this file inside the Editor [Assets/Resources/VuforiaConfiguration.asset](Assets/Resources/VuforiaConfiguration.asset)
## Playing
- The Image Target for starting the game can be found at [Assets/SamplesResources/InstantImageTarget/Astronaut.jpg](Assets/SamplesResources/InstantImageTarget/Astronaut.jpg)
- Some rules were adjusted to make the game easier and playable.

Enjoy!
